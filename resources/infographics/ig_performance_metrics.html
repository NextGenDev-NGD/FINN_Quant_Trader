Performance Metrics Reference

Sharpe ratio, max drawdown, win rate, expectancy, IC — definitions, formulas, thresholds
Tier 2 · FINN-Specific

FINN framing rule: A strategy has a win rate. A strategy has a Sharpe ratio. These are not the same thing, and neither alone tells you whether the strategy is worth trading. A complete performance picture requires at least: Sharpe (risk-adjusted return), max drawdown (survivability), expectancy (per-trade profit expectation), and IC (signal quality). Know what each measures before you cite it.

Core Metrics — Definitions, Formulas, Thresholds
Sharpe Ratio
SR
SR = (R_p − R_f) / σ_p R_p = portfolio return (annualized) R_f = risk-free rate (~5% currently) σ_p = annualized std dev of returns
Measures return per unit of risk. A Sharpe of 1.0 means you earned 1× your volatility above the risk-free rate. Does not distinguish between upside and downside volatility.
Thresholds
> 2.0Exceptional — institutional qualityStrong signal
1.0 – 2.0Good — tradeable edgeTrade it
0.5 – 1.0Marginal — refine before allocatingRefine
< 0.5Not worth the risk — rebuildReject
Maximum Drawdown
MDD
MDD = (Trough − Peak) / Peak Peak = highest portfolio value before trough Trough = lowest portfolio value after peak Expressed as a negative percentage
Measures the worst peak-to-trough loss. Tells you the most pain you would have endured staying in the strategy. More important than average return for position sizing decisions.
Thresholds
< 10%Excellent capital preservationContinue
10% – 20%Acceptable — monitor closelyWatch
20% – 35%High — reduce position sizeReduce
> 35%Catastrophic — halt tradingHalt
Win Rate + Expectancy
WR / E
WR = Winning trades / Total trades E = (WR × Avg_Win) − (LR × Avg_Loss) LR = 1 − WR (loss rate) E > 0 required for profitable strategy
Win rate alone is meaningless. A strategy with 40% WR and 3:1 R:R outperforms a 70% WR strategy with 0.5:1 R:R. Expectancy combines both into a single per-trade profit expectation.
Expectancy Thresholds
> $1.50/RStrong edge per unit of riskScale up
$0.50–$1.50/RTradeable — consistent execution neededTrade it
$0.01–$0.50/RSlim edge — transaction costs matterRefine
≤ $0Negative expectancy — losing strategyReject
Information Coefficient
IC
IC = Spearman_correlation( signal_today, forward_return_N_days ) Ranges: −1.0 to +1.0 IC = 0 → signal has no predictive power
FINN's primary signal quality measure. Measures rank correlation between signal values and subsequent returns. An IC of 0.10 is considered strong in professional quant work — most real edges are IC 0.03–0.08.
IC Thresholds (FINN Standards)
> 0.10Exceptional — publish or scaleStrong
0.05 – 0.10Good — tradeable signal qualityTrade
0.03 – 0.05Weak — needs combination with other signalsCombine
< 0.03No edge — rebuildReject
Python Reference — Compute All 4 Metrics
performance_metrics.py — FINN standard calculation block
Python 3.11
import numpy as np
import pandas as pd
from scipy import stats

# --- Sharpe Ratio ---
def sharpe_ratio(returns, risk_free_annual=0.05):
    rf_daily = risk_free_annual / 252
    excess   = returns - rf_daily
    return (excess.mean() / excess.std()) * np.sqrt(252)

# --- Max Drawdown ---
def max_drawdown(equity_curve):
    roll_max = equity_curve.cummax()
    drawdown = (equity_curve - roll_max) / roll_max
    return drawdown.min()

# --- Expectancy ---
def expectancy(trade_pnl):
    wins   = trade_pnl[trade_pnl > 0]
    losses = trade_pnl[trade_pnl < 0]
    wr     = len(wins) / len(trade_pnl)
    avg_w  = wins.mean()  if len(wins) > 0 else 0
    avg_l  = losses.mean() if len(losses) > 0 else 0
    return wr * avg_w + (1 - wr) * avg_l

# --- Information Coefficient ---
def information_coefficient(signal, forward_return):
    ic, _ = stats.spearmanr(signal.dropna(), forward_return.dropna())
    return ic

# --- Summary ---
def performance_summary(returns, equity, trade_pnl, signal, fwd_ret):
    return {
        'sharpe':    round(sharpe_ratio(returns), 3),
        'max_dd':    round(max_drawdown(equity) * 100, 2),
        'expectancy': round(expectancy(trade_pnl), 4),
        'ic':        round(information_coefficient(signal, fwd_ret), 4),
    }

FINN

"Win rate is a vanity metric. I've seen 70% win-rate strategies blow up and 35% win-rate strategies compound for years. What matters is the combination of Sharpe, drawdown, and expectancy — and none of those matter if the IC says the signal has no predictive power to begin with. Start with IC. Everything else follows."
