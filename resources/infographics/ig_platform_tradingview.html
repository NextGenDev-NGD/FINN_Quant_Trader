Chart Types & Timeframes
Chart Types
  • Candlestick — FINN standard. Shows OHLC and candle patterns.
  • Bar (OHLC) — Same data, different visual. Preferred by some discretionary traders.
  • Line — Close prices only. Useful for identifying trends without noise.
  • Heikin Ashi — Smoothed candles for trend identification. See Handbook §2.7.
  • Renko / Kagi — Non-time-based. Informational only in FINN.

FINN default: Candlestick for analysis; Heikin Ashi as a secondary view for trend confirmation.

Key Timeframes
  • Weekly (1W) — Major S&R; long-term trend context
  • Daily (1D) — Primary FINN analysis timeframe
  • 4-Hour (4H) — Intermediate structure; MTF context
  • 1-Hour (1H) — Entry setup identification
  • 15-Minute (15M) — Entry timing and stop placement
  • 5-Minute (5M) — Execution detail (advanced)

FINN MTF sequence: Start daily → 4H → 1H → 15M. Never start at the lowest timeframe. See Handbook §6.3.

Useful Symbols
  • SPY — S&P 500 ETF (daily benchmark)
  • QQQ — Nasdaq 100 ETF
  • VIX — CBOE Volatility Index
  • ES1! — S&P 500 futures (continuous)
  • NQ1! — Nasdaq 100 futures
  • DXY — US Dollar Index
  • TLT — 20+ Year Treasury ETF (yield proxy)
  • GLD — Gold ETF
FINN Standard Indicator Setup
Price Overlays
  • SMA 20 — Short-term trend; pullback level in uptrends
  • SMA 50 — Intermediate trend; major S&R in most setups
  • SMA 200 — Long-term regime filter (above = bull, below = bear)
  • VWAP — Intraday fair value; institutional reference price
  • Bollinger Bands (20, 2) — Volatility envelope; mean-reversion signal

Add via: Indicators → Search by name → click to add

Sub-Panel Indicators
  • RSI (14) — Momentum oscillator; overbought/oversold reference
  • MACD (12,26,9) — Trend momentum and crossover signals
  • Volume — Always on. Color-coded: green = up volume, red = down volume
  • ATR (14) — Volatility measurement; used for stop sizing
  • Stochastic RSI — Faster oscillator for short-term momentum

Sub-panels appear below the chart. Drag the divider to resize.

Save as Template

Once your indicator set is configured, save it as a template so every new chart loads with the same setup:

Chart Settings → Templates → Save as Template → "FINN Standard"

Apply to any new symbol: click the template name in the template dropdown.

Layout vs. Template: A template saves indicators only. A layout saves the full workspace (chart + watchlist + panels). Save both.

Setting Alerts — The Right Way
Price Level Alerts

Set an alert when price crosses a key level so you don't have to watch the chart continuously:

Method 1 (fastest): Right-click on the chart at the price level → Add Alert at [price]

Method 2: Click the clock icon (Alerts panel) → Create Alert → set symbol, condition, price

Conditions available: Crossing, crossing up, crossing down, greater than, less than.

Notification: Browser popup, email, or TradingView app push notification. Configure in Alert settings → Notifications.

Indicator Alerts

Alert when an indicator crosses a threshold — not just price:

  • RSI crosses 70 — overbought entry into overbought zone
  • RSI crosses 30 — oversold; potential reversal watch
  • MACD crosses signal line — momentum shift
  • Price crosses SMA 50 — trend change alert

Set via: Alerts → Create Alert → Condition → [Indicator name] → [Crossing direction] → [Value]

FINN use: Set alerts on your watchlist tickers at key S&R levels and SMA crossovers each morning. Do not watch charts continuously — let alerts bring you back.

Watchlists & Multi-Chart Layouts
FINN Watchlist Structure
  • Market Dashboard — SPY, QQQ, VIX, DXY, TLT, GLD, ES1!, NQ1!
  • Active Setups — 5–8 tickers with patterns forming; updated daily
  • Sector ETFs — XLK, XLF, XLE, XLV, XLI, XLY, XLP, XLU, XLB, XLRE, XLC
  • On Watch — Tickers approaching key levels; not yet a setup

Manage via: Watchlist panel → + → New watchlist → Name it

Multi-Chart Layout

View multiple tickers simultaneously using TradingView's layout feature:

Free plan: 1 chart per layout (single view)

Pro plan: Up to 4 charts per layout (2×2 grid)

To set up: Layout button (top right) → Select grid → assign each panel a symbol and timeframe

FINN morning review setup: 4-chart layout with SPY (daily), QQQ (daily), VIX (daily), and your top active setup (4H).

Pine Script (Awareness)

TradingView's proprietary scripting language for building custom indicators and strategy backtests directly in the platform.

  • Can code custom signals (e.g., "highlight candles where RSI < 30 and volume > 1.5× average")
  • Can run simple backtests with entry/exit logic
  • Results are visual — not as rigorous as Python backtests

FINN position: Pine Script is useful for quick visual exploration. All formal backtesting in FINN uses Python + pandas for statistical rigor.

TradingView is your analysis environment — not your execution platform. You draw levels, set alerts, and identify setups here. You execute orders in Webull (paper) or your live broker. Never confuse the two tools.

Free vs. Pro: The free tier is sufficient for FINN. You get 1 chart layout, 3 indicators per chart, and limited alerts. If you find yourself hitting these limits consistently, Pro ($14.95/month) unlocks 4-chart layouts, unlimited indicators, and more alerts. Not required for the course.
